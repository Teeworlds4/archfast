# ArchFast
## Install Archlinux
Installing:
```
pacman -Sy
archinstall
```
Recomended setup:
```
DE*: XFCE4
Audio: Pipewire
Kernel: linux-zen
Packages: linux-firmware base base-devel efibootmgr grub git curl lightdm-webkit2-greeter mesa htop bashtop neofetch p7zip vlc firefox steam discord 
```
## Post-Install
```
git clone https://github.com/Teeworlds4/archfast.git
cd archfast
bash aur.sh
```
